# 🎨 Insurance Assistant: Premium Frontend

This directory contains the high-end React frontend for the Insurance Assistant, built with a focus on **Premium UX/UI**, **Glassmorphism**, and **Seamless AI Integration**.

---

## 🚀 Teck Stack
- **Framework:** [React 18](https://reactjs.org/) (via [Vite](https://vitejs.dev/))
- **Styling:** [TailwindCSS](https://tailwindcss.com/) + Custom CSS Glassmorphism
- **Routing:** [React Router Dom v6](https://reactrouter.com/)
- **State & API:** Context API & [Axios](https://axios-http.com/)
- **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit) (Headings) & [Inter](https://fonts.google.com/specimen/Inter) (Body)

## ✨ UI Highlights
- **Premium Design System:** Centralized tokens in `index.css` for consistent glass effects and professional shadows.
- **Micro-Animations:** Powered by Tailwind and custom keyframes (e.g., `animate-fade-in`).
- **Conditional Layouts:** SaaS-style sidebar for authenticated users and elegant landing sections for visitors.

## 📂 Key Directories
- `src/components/`: Reusable premium components (Navbar, Chatbot, Modals).
- `src/pages/`: Main views (Home, Dashboard, Admin, Policies, Login).
- `src/context/`: Auth state management and JWT handling.
- `src/api/`: Axios interceptors for authenticated backend communication.

## 🛠️ Development
### Install Dependencies
```bash
npm install
```
### Start Dev Server
```bash
npm run dev
```
